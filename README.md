# 🎬 TMDB Movie App

A React-based Movie Application built using the TMDB API.

## 🚀 Live Demo

👉 https://tmdb-react-project.vercel.app/

## 📂 GitHub Repository

👉 https://github.com/ohitzdhanush/TMDB

## ✨ Features

- Search Movies
- Popular Movies
- Upcoming Movies
- Movie Details Page
- Trailer Button
- Pagination
- Debounce Search
- Responsive UI

## 🛠️ Technologies Used

- React.js
- React Router DOM
- Axios
- SCSS
- TMDB API
- Vercel

## ▶️ Run Locally

```bash
npm install
npm run dev
```
